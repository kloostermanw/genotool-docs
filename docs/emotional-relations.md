# Emotional Relations

Emotional relations describe the interpersonal dynamics between any two persons in a genogram. Unlike [family relations](relations.md), emotional relations are not about formal partnerships but about the quality of the connection between individuals.

## Adding an Emotional Relation

To add an emotional relation:

1. Select a person in the drawing.
2. In the detail panel, navigate to the **Emotional Relations** section.
3. Select the other person and choose the type of emotional relation.

The emotional relation will appear as a distinct line style between the two persons.

## Emotional Relation Types

### Positive Relations

| Type | Description |
|------|-------------|
| **Normal** | A standard, unremarkable relationship. |
| **Harmony** | A balanced, harmonious connection. |
| **Friendship** | A warm, platonic bond. |
| **Best Friends** | A close, strong friendship. |
| **Love** | A deep feeling of love between two persons. |
| **In Love** | A state of romantic attraction. |
| **Emotional Connection** | A meaningful emotional bond. |
| **Fan** | An admiring or fan-like relationship. |
| **Limerence** | An intense, involuntary romantic attraction. |

### Negative Relations

| Type | Description |
|------|-------------|
| **Discord** | Disagreement or conflict in the relationship. |
| **Hate** | Deep animosity between the persons. |
| **Distrust** | Lack of trust between the persons. |
| **Hostile** | Active antagonism and aggression. |

### Distance & Cutoff

| Type | Description |
|------|-------------|
| **Indifferent** | Emotional indifference between the persons. |
| **Distant** | An emotionally distant relationship. |
| **Cutoff Estranged** | The relationship has been cut off; the persons are estranged. |
| **Cutoff Repaired** | A previously cut-off relationship that has been restored. |
| **Never Met** | The two persons have never met. |

### Hostile Combinations

| Type | Description |
|------|-------------|
| **Distant Hostile** | A relationship that is both distant and hostile. |
| **Close Hostile** | A close relationship with hostile dynamics. |
| **Fused Hostile** | An enmeshed relationship with hostile elements. |

### Fused Relations

| Type | Description |
|------|-------------|
| **Fused** | An enmeshed or overly dependent relationship. |

### Violence

| Type | Description |
|------|-------------|
| **Violence** | Physical violence is present in the relationship. |
| **Distant Violence** | Violence occurring in an otherwise distant relationship. |
| **Close Violence** | Violence within a close relationship. |
| **Fused Violence** | Violence within an enmeshed relationship. |

### Abuse

| Type | Description |
|------|-------------|
| **Abuse** | A pattern of abusive behavior. |
| **Physical Abuse** | Physical harm or violence. |
| **Emotional Abuse** | Psychological or emotional harm. |
| **Sexual Abuse** | Sexual abuse or exploitation. |
| **Neglect Abuse** | Abuse through neglect of care. |

### Behavioral Patterns

| Type | Description |
|------|-------------|
| **Manipulative** | One person uses manipulation to influence the other. |
| **Controlling** | One person exerts control over the other. |
| **Jealous** | Jealousy characterizes the relationship. |
| **Focused On** | One person is particularly focused on the other. |
| **Focused On Negatively** | One person is negatively focused on or preoccupied with the other. |
