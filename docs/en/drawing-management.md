# Drawing Management

A drawing is a single genogram in GenoTool. You can create, rename, duplicate, and delete drawings from the drawing overview page.

## Creating a New Drawing

1. Navigate to the drawing overview page.
2. Click **New Drawing**.
3. Enter a name for the drawing (maximum 100 characters). The name must be unique within your team.
4. Click **Save**.

A new drawing opens with a blank canvas containing one starting person.

## Browsing Drawings

The drawing overview page shows all drawings within your current team in a table with the following columns:

- **Name** - The drawing name
- **Created by** - Who created the drawing
- **Updated by** - Who last modified the drawing
- **Actions** - Available actions (edit, duplicate, export, delete)

You can sort the table by clicking on the column headers and use the search functionality to find specific drawings.

## Renaming a Drawing

1. On the drawing overview page, click the **edit** icon next to the drawing you want to rename.
2. Enter the new name (maximum 100 characters, must be unique within your team).
3. Click **Save**.

## Duplicating a Drawing

To create a copy of an existing drawing:

1. On the drawing overview page, click the **duplicate** icon next to the drawing.
2. Enter a name for the duplicate.
3. Click **Save**.

The duplicate contains a complete copy of the original, including all persons, relations, groups, settings, and positions.

## Deleting a Drawing

1. On the drawing overview page, click the **delete** icon next to the drawing.
2. Confirm the deletion in the dialog that appears.

!!! warning
    Deleting a drawing cannot be undone. Make sure to export your drawing first if you may need it later.
