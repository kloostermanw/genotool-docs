# Legend

The legend provides an overview of all symbols used in your genogram. It is generated automatically based on the elements present in your drawing.

## Show / Hide Legend

The legend is hidden by default. To display it:

1. Go to `Settings > Legend`.
2. Toggle **Show Legend** on.

The legend will appear on the canvas showing all symbols currently used in your drawing.

## Legend Position

You can place the legend in one of the following positions on the canvas:

- **Left Top**
- **Right Top**
- **Left Bottom**
- **Right Bottom**
- **Top** (centered)
- **Center**

Change the position in `Settings > Legend` using the **Legend Position** dropdown.

## Legend Orientation

The legend can be displayed in two orientations:

- **Horizontal** - Items are arranged side by side
- **Vertical** - Items are stacked on top of each other

Change the orientation in `Settings > Legend` using the **Legend Orientation** dropdown.

## Legend Font Size

The font size of the legend text can be adjusted independently from other text in the drawing. Go to `Settings > Legend` to change the font size.

See [Fonts](fonts.md) for more information on customizing font sizes.
